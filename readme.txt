Plugin to add Ratapay as Woocommerce Payment Gateway and more.

== Description ==

Ratapay is an official plugin to integrate WooCommerce powered WordPress site with Ratapay System. By Using this plugin you can use Ratapay as Payment Gateway for your shop. There are also some features which will help you integrate better with the splitter system provided by Ratapay such as:

* Paylink Management: Manage Ratapay Paylink directly from Wordpress admin page
* Splitter Management: Define splitter template and apply it on any product

You can go to [Ratapay](https://ratapay.co.id/ "Ratapay Home") to find more info about Ratapay and its features.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ratapay` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Ratapay->Setting screen to configure the plugin
1. Fill in your Merchant ID and API Key, you can optionally configure the other options to
1. Click Submit
1. Go to WooCommerce Payment Setting Menu
1. Activate Ratapay
1. Your're good to go with the basic setup

== Frequently Asked Questions ==

= Where can I get the Merchant ID and API Key? =

It will be sent through your email after your account successfully verified.

= What is sandbox Mode? =

It is a mode for you to test the integration without cluttering your real transaction.

= Where can I get the Merchant ID and API Key for sandbox mode? =

It will be sent through your email when you go to sanbox from ratapay app accout page.
